# USB_Oscilloscope PCB

ペンタイプオシロスコープ用の KiCad 基板設計データです。回路の基本はDroidOscilloを踏襲しています。

このリポジトリには、回路図、PCB レイアウト、カスタムシンボル、カスタムフットプリント、3D モデル、製造用データを含みます。

## 概要

- KiCad プロジェクト: `USB_Oscilloscope.kicad_pro`
- 回路図: `USB_Oscilloscope.kicad_sch`
- PCB レイアウト: `USB_Oscilloscope.kicad_pcb`
- カスタムライブラリ: `ps_lib/`
